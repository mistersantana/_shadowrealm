# Zero2Hero Database Project
The purpose of this page is to document the expectations and technical goals for the Zero2Hero course project.

# Employee Database
You’ll build an Employee Database program with the following features:

Store information about employees:
- Name
- Address
- Hours Worked
- Add employees to the database
- Update employees in the database
- Store all this information to a file
- Load and edit the database file
- Search the database by:
- Name
- Address
- Hours Worked
- This project will force you to apply everything you've learned and help you start thinking like a systems programmer.

Expectations:
- You are expected to write the database in C using your own code.
- You can use any standard library functions taught in this course.
- You should aim for clean, readable code with comments explaining complex parts.
- You’ll be given starter templates for struct definitions and some helper functions.
